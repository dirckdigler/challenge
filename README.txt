


Challenge is a module that consume theMoviedb webservice, perfect to populate a page about movies.
It's not necessary to turn on others modules different to "challenge"

-- GETTING STARTED --

1. Download and install a Drupal core
2. Create vhost to previously installed Drupal
3. Overwrite the modules folder according to the .zip that I shared. You must see this structure modules/custom/challenge
4. Turn on the module
3. The module automatically place various blocks at frontpage and create differents pages.
4. Ready to navigate the website.
